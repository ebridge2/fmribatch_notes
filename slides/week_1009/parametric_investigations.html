Model

Data

  • Given: $n$ samples of data-tuples, $\{(g_i, y_i, z_i)\}_{i=1}^n$
  • $g_i$ is a graph $(E, V, w)$ where $w(v_i, v_j) = w_{ij}$
  • a set of fixed-effects $y_i$ that we anticipate will be signal
  • a set of random-effects $z_i$ that we anticipate will be noise due to batch effects

Task

  • Characterize the composition of signal to batch-effect present

Random-Effects Model

$w_{ij,s,x,n,d} = \sum_{k=1}^K\beta_{ij,s, x, n, d}^{(k)} X_{ij, s, x, n, d}^{(k)} + \epsilon_{ij} + \mu_{ij}$

  • Random effect $k$ with values $X^{(k)}_{ij, \cdot}$ and coefs $\beta_{ij, \cdot}$
  • Effects modelled: subject $s$, sex $x$, session $n$, dataset $d$
  • mean weight for edge $i, j$ $\mu_{ij}$
  • residual for edge $i, j$ $\epsilon_{ij}$

Assumptions

  • $Var(w_{ij, s,x,n,d}) = \sum_{k=1}^K Var(X_{id, s,x,l,d}^{(k)}) + Var(\epsilon)$
  • That is, the variance of the response is the sum of the variance of the components
  • $\rightarrow Cov(X_{s,x,l,d}^{(k)}, X_{s,x,l,d}^{(l)}) = 0$

Evaluation

  • percent variance for $k$: $\frac{Var(X_{ij, s,x,l,d}^{(k)})}{Var(X_{ij, s,x,n,d})}$
  • Dependability: $\phi_{UV}(X_{ij}) = \frac{Var(subject)}{\sum_{k=1}^K \frac{Var(X_{ij, s,x,n,d}^{(k)})}{n_{s, x, n, d}}}$
